# Generator de Coduri QR în Python

Aplicație utilitară în Python dezvoltată pentru generarea automatizată a unui cod QR ce trimite direct către o platformă de streaming muzical. Proiectul pune în practică utilizarea și configurarea librăriilor externe (precum `qrcode`), manipularea fișierelor de tip imagine și automatizarea sarcinilor de partajare media.

# Ce am învățat dezvoltând acest script:
* Integrarea ecosistemului Python: Cum să caut, să instalez prin `pip` și să folosesc corect o librărie externă citindu-i documentația oficială.
* Automatizare practică: Cum poți transforma o sarcină repetitivă (deschiderea unui browser, căutarea unei piese, copierea link-ului) într-o singură imagine QR gata de scanat și trimis prietenilor.
* Gândire orientată spre utilitate: Chiar dacă este un script mic, rezolvă o problemă reală de partajare rapidă a conținutului media.
